# 🏗 Shop Game - Web Builder Monorepo

Dự án Shop bán tài khoản game tích hợp hệ thống Web Builder kéo thả hiện đại.

## 📁 Cấu trúc thư mục

```text
web-builder/
├── frontend/         # React App (Vite + TS + Tailwind)
├── backend/          # Django REST Framework (Python)
├── docs/             # Tài liệu hệ thống chi tiết
├── agents/           # Cấu hình AI Agent (Skills, Rules)
└── docker/           # Cấu hình Docker & Nginx
```

## 🚀 Hướng dẫn bắt đầu

### 1. Tài liệu chi tiết
Xem tài liệu kiến trúc tại: [docs/ARCHITECTURE.md](./docs/ARCHITECTURE.md)

### 2. Chạy Frontend (Dev)
```bash
cd frontend
npm install
npm run dev
```

### 3. Chạy Backend (Dev)
(Đang cập nhật cấu hình Django)
